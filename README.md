# ROTAX — Körper-Nachzeichner

A retro 3D puzzle game inspired by a 1992 Turbo Pascal original.

A 3D shape (Urkörper) is shown on the left. A rotated version (Dünnkörper) appears on the right — trace all its visible edges by clicking them.

**Play:** open `index.html` in any browser. No build step, no dependencies.

## Difficulty
- **Leicht** — 1–2 connected cubes
- **Mittel** — 3–7 connected cubes
- **Schwer** — 3–6 cubes, any configuration
- **Wahnsinn** — 3×3×3 grid, 5–18 cubes, fully random placement

## Original sources
The `original_rotax/` folder contains the original 1992 Turbo Pascal source files the project is based on.
